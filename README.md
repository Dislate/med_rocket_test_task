# med_rocket_test_task
This is simple implement reporting system with getting data from outside API's
```
python3 -m venv venv
```
## Activate virtual environment
For Linux
```
. venv/bin/activate
```
or Windows
```
. venv/Scripts/activate
```


## Install dependencies
```
pip3 install -r requirements.txt
```

## Run script
```
python3 main.py
```
